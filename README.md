# 这是一个测试使用GitHub的项目

## 新建版本库,GitHub给我们的帮助

```shell
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/zhouqinfengqq/test_2020.git
git push -u origin master
```
